BatchPaste

  • What is it?

    • A macro for Google Spreadsheets written in Google Apps Script.

  • What is it for?

    • Calculating Differential equations

  • How does it work?

    • You type in constants that proceed from one variable to another. The macro will convert your input into a differential equation and will display your changes on a graph.

  • How do I use it?

    • Open Google Spreadsheet (https://docs.google.com/spreadsheet) and go to Tools > Script Editor.

    • Copy and paste the code, then hit save (name it whatever you want).

    • Go back to the Spreadsheet and reload the page.

    • Select “Prototype Actions” from the upper ribbon and go to “refresh backend”.

    • Authorize the macro and then select “refresh backend” again (then wait 1 min).

    • Select “refresh frontend” and wait 1 min.

    • Once these steps are complete the macro should be ready for use.
